# ₴ ГРОШ — крипто-радар

Бот-сканер перспективных монет: каждый час собирает данные с CoinGecko и DexScreener, считает «скор потенциала» и риск-метрики (волатильность, VaR 95%, просадка, бета к BTC, слиппедж) и показывает результат на сайте-дашборде.

**Без API-ключей, без сервера, бесплатно** — работает на GitHub Actions + GitHub Pages.

## Что внутри

- `scanner.py` — сканер: тренды CoinGecko, топ-750 монет, продвигаемые токены DexScreener → `docs/data.json`
- `docs/index.html` — дашборд (тёмная тема): ⭐ Топ по скору, 🔥 Тренды, 🚀 Движения 24ч, 💎 Гемы, 🆕 DEX-хайп
- `.github/workflows/scan.yml` — автозапуск каждый час

## Как опубликовать (один раз)

1. Создай на GitHub новый **публичный** репозиторий, например `crypto-radar`
   (аккаунт: martvitaliy21-stack → New repository).
2. Загрузи туда все файлы этой папки. Проще всего с терминала:
   ```bash
   cd ~/claude/crypto-scanner
   git remote add origin https://github.com/martvitaliy21-stack/crypto-radar.git
   git push -u origin main
   ```
   (или через веб: Add file → Upload files — но тогда папку `.github/workflows` нужно создать вручную через Create new file: `.github/workflows/scan.yml`)
3. В настройках репозитория: **Settings → Pages → Branch: `main`, папка `/docs` → Save**.
4. Во вкладке **Actions** разреши workflows, если GitHub спросит.
5. Через пару минут сайт будет доступен:
   `https://martvitaliy21-stack.github.io/crypto-radar/`

Дальше всё само: каждый час в :05 Actions запускает сканер и обновляет данные на сайте.

## Локальный запуск

```bash
python3 scanner.py                       # обновить данные
python3 -m http.server 8765 -d docs      # открыть http://localhost:8765
```

## Как читать скор (0–100)

- **Объём/капитализация** (до 30) — насколько активно монету торгуют
- **Моментум** (до 50) — рост за 1ч / 24ч / 7д
- **Тренды CoinGecko** (+20) — монета в топе интереса

⚠️ Это автоматическая оценка интереса рынка, **не финансовый совет**. Высокий скор часто = высокая волатильность. Всегда проверяй проект сам (DYOR), особенно вкладку DEX-хайп — там платное продвижение и много скама.
